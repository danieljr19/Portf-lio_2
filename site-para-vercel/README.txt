INSTRUÇÕES PARA DEPLOY NO NETLIFY:

1. Faça o upload da pasta "public" para o Netlify
2. Esta pasta contém todos os arquivos necessários para o seu site:
   - index.html (arquivo principal)
   - pasta assets (CSS e JavaScript)
   - _redirects (configuração de rotas)

IMPORTANTE: Você deve fazer o upload da pasta "public" e não apenas dos arquivos individuais.
