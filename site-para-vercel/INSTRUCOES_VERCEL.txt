INSTRUÇÕES PARA DEPLOY NO VERCEL:

Existem dois métodos para fazer o deploy no Vercel:

MÉTODO 1 - DEPLOY DIRETO:
1. Acesse https://vercel.com e faça login ou crie uma conta
2. Clique em "Add New" > "Project"
3. Selecione a opção "Upload" na barra lateral
4. Arraste e solte a pasta "public" deste pacote
5. Clique em "Deploy"

MÉTODO 2 - DEPLOY VIA GITHUB:
1. Faça upload deste projeto para um repositório GitHub
2. Acesse https://vercel.com e faça login ou crie uma conta
3. Clique em "Add New" > "Project"
4. Importe o repositório GitHub
5. Configure estas opções:
   - Framework Preset: Vite
   - Root Directory: . (ponto)
   - Build Command: npm run build
   - Output Directory: dist/public
6. Clique em "Deploy"

IMPORTANTE: O arquivo vercel.json já está configurado para lidar com as rotas da SPA corretamente.
